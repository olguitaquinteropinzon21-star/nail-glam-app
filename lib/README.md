# 💅 Nail Glam

Aplicación móvil desarrollada en Flutter para la gestión de servicios de manicure, diseños de uñas y reservas de citas.

## Características

- Inicio de sesión
- Registro de usuarios
- Catálogo de servicios
- Diseños de uñas
- Reserva de citas
- Gestión de citas
- Perfil de usuario

## Tecnologías utilizadas

- Flutter
- Dart
- Material Design
